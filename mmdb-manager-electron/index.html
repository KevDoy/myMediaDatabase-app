        Import

        Import your library from various digital platforms and services.

        YouTube Import purchased movies from YouTube
        ›
        1. In Chrome, navigate to https://www.youtube.com/feed/storefront
        2. Click Purchased → View All
        3. Right-click → Save As to download the HTML file
        4. Drag the file below or click "Choose file"

        Drag & drop the saved HTML file here

        Movies Anywhere Import your Movies Anywhere digital library
        ›
        1. In Chrome, open https://moviesanywhere.com/my-movies
        2. Right-click → Save As to download the HTML file
        3. Drag the file below or click "Choose file"

        Drag & drop the saved HTML file here

        Fandango at Home Import your Fandango at Home (Vudu) library
        ›
        1. In Chrome, open https://athome.fandango.com/content/browse/mymovies
        2. Right-click → Save As to download the HTML file
        3. Drag the file below or click "Choose file"

        Drag & drop the saved HTML file here

        Letterboxd Ratings Import ratings and reviews from Letterboxd
        ›
        1. Go to https://letterboxd.com/settings/data/ and export your data
        2. You'll receive a .zip file
        3. Drag the .zip file below

        Drag & drop your Letterboxd export .zip here

        CSV / Text File Import game titles from a CSV or one-per-line text file
        ›
        1. Prepare a .csv with a "Name" or "Title" column, or a plain .txt with one game per line
        2. Optionally include a "Platform" column
        3. Drag the file below or click "Choose file"

        Drag & drop the file here
